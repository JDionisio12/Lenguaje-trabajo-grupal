#Lista de elementos que se rompen al reducir la pantalla

1. El body al tener min-width:1200px aparece scroll horizontal.

2. El main y aside usan float y tienen ancho fijo, provoca desbordamiento.

3. La imagen tiene tamaño fijo.

4. El menú horizontal no cabe bien en móvil.

5. El footer tiene width:1200px, provoca scroll horizontal en pantallas pequeñas.
