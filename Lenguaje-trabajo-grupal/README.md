
# 🌐 Proyecto: Lenguaje de Marcas - Trabajo Grupal

Bienvenidos al repositorio de nuestro proyecto web. Este sitio ha sido desarrollado como parte de la asignatura de **Lenguaje de Marcas**, enfocado en el uso de HTML5, CSS3 y diseño adaptable (Responsive Design).

## 🚀 Demo en Vivo
Puedes ver la página funcionando aquí:
👉 [https://jdionisio12.github.io/Lenguaje-trabajo-grupal/](https://jdionisio12.github.io/Lenguaje-trabajo-grupal/)

---

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica del contenido.
* **CSS3:** Estilos, diseño de Layout y animaciones.
* **Media Queries:** Para asegurar que la web se vea bien en móviles, tablets y ordenadores.
* **GitHub Pages:** Para el despliegue y hosting de la web.

## 📂 Estructura del Proyecto
* `index.html`: Archivo principal de la página.
* `styles.css`: Hoja de estilos con todo el diseño visual.
